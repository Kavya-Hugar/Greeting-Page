# Greeting Web App

A modern, responsive greeting web application with beautiful animations and interactive features.

## 🚀 Live Demo

Visit the live demo: [https://kavya-hugar.github.io/Greeting-Page/](https://kavya-hugar.github.io/Greeting-Page/)

## ✨ Features

- **Modern UI**: Glassmorphism design with gradient backgrounds
- **Dynamic Greetings**: Personalized greetings with smooth animations
- **Random Animations**: Three unique celebration effects:
  - 🎊 Confetti fall
  - 🎉 Party popper burst
  - ✨ Glowing burst effects
- **Responsive Design**: Works perfectly on all devices
- **Cross-browser Compatible**: Runs smoothly on all modern browsers
- **Error Handling**: Robust error handling and fallbacks

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with animations and transitions
- **JavaScript ES6+**: Interactive functionality
- **Node.js**: Development server
- **GitHub Pages**: Hosting

## 📱 How to Use

1. Enter your name in the input field
2. Click the "Greet" button or press Enter
3. Watch the random animation and personalized greeting!

## 🚀 Local Development

### Option 1: Using Node.js (Recommended)
```bash
# Clone the repository
git clone https://github.com/Kavya-Hugar/Greeting-Page.git
cd Greeting-Page

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Option 2: Using Python
```bash
# Start Python server
npm run serve
```

### Option 3: Direct File Opening
Simply open `index.html` in your browser.

## 🎨 Animations

The app features three randomly triggered animations:

1. **Confetti**: Colorful falling confetti particles
2. **Party Popper**: Explosive burst effect from center
3. **Glowing Burst**: Random glowing orbs that expand and fade

Each animation is properly managed to prevent overlap and ensure smooth performance.
